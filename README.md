#AVR I/O Macros  
  
---  
**Description:**  
Macros to control the I/O hardware of AVR microcontrollers in a clean and elegant way. The macros resemble those of an Arduino.  
Macros list:  
* PinMode(Pin, Mode),  
* DigitalWrite(Pin, Level),  
* DigitalRead(Pin),  
* PinModeToggle(Pin),  
* DigitalLevelToggle(Pin).  
  
---  

**Memory usage:**  
+ Program: 0 bytes  
+ Data: 0 bytes  
  
**Compiler:**  
AVR-GCC  
  
**Optimization Level:**  
Optimize (-O1)
